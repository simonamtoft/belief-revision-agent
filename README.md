# Belief Revision Agent
A belief revision engine made for the Intro to AI course on DTU.





## Program Syntax

| Logic | Program Input |
| ----- | ----- |
| a $\lor$ b | a \| b | 
| a $\land$ b | a & b | 
| $\lnot$ a | ~a| 
| a $\to$ b | a >> b|
| a $\leftrightarrow$ b | a >> b & a << b |
